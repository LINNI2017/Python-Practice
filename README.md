# [1732](https://leetcode.com/problems/find-the-highest-altitude/)
```python
def main(gain):
   cur = 0
   maxi = 0
   for i in gain:
      cur += i
      maxi = max(maxi, cur)
   return maxi
```
#### Assumption: N = the number of elements in gain list
#### Complexity: runtime = O(N), space = O(1)
