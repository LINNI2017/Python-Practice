# [326](https://leetcode.com/problems/power-of-three/)
```python
def main(n):
  if n == 0:
    return False
  while n % 3 == 0:
    n /= 3
  return n == 1
```
#### Assumption: N = the size of the given number
#### Complexity: runtime = O(logN), space = O(1)
